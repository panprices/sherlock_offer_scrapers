# Sherlock Offer Scrapers

[![Code style: black](https://img.shields.io/badge/code%20style-black-000000.svg)](https://github.com/psf/black)

A collection of scrapers to find offers from multiple sources.

## Run tests
To run a sample test of a scraper, first enable virtual environment with `pipenv shell`, then run `python demo.py {scraper-name}`
For example:
```bash
$ python demo.py idealo
```
